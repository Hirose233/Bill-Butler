# Bill-Butler
This is a project built using Python and React for personal budgeting and income/expense tracking.

## Technology stack
Back-end：Python + FastAPI + SQLAlchemy + SQLite
Front-end：React + Vite + Tailwind CSS + Axios

## How to run it
1. cd backend
2. pip install -r requirements.txt
3. python -m uvicorn main:app --reload
## Front end
1. cd frontend
2. npm install
3. npm run dev
